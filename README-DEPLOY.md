# HONGJU Portfolio — 배포용 파일

이 폴더 전체가 웹사이트입니다. GitHub Pages에 이 폴더의 내용을 그대로 올리세요.

- `index.html` — 공개 웹사이트의 첫 화면
- `images/` — 프로젝트 이미지
- `portfolio.html` — 기존 로컬 주소를 위한 자동 연결 페이지

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더 안의 `index.html`, `images` 폴더, `portfolio.html`을 저장소 최상단에 올립니다.
3. 저장소의 **Settings → Pages**에서 `main` 브랜치와 `/(root)`를 선택합니다.
4. 발행된 주소(`https://내아이디.github.io/저장소이름/`)를 사용합니다.

프로젝트 내용이나 이미지를 바꾼 뒤 같은 파일을 저장소에 다시 올리면 공개 사이트도 업데이트됩니다.
